# VC 04 — Repurposing-Tools: Ein Video → Alle Plattformen
**Tools:** Opus Clip, CapCut, Buffer / Publer
**Relevanz für MAJER:** Hoch — Repurposing ist die Grundlage der "alle Plattformen gleichzeitig"-Strategie

---

## Kernaussage
Repurposing bedeutet: Ein Longform-Video (10–15 Min) wird automatisch in 3–7 Short-Clips zerschnitten, mit Captions versehen, für 9:16 formatiert und auf TikTok, YouTube Shorts, Instagram Reels gleichzeitig gepostet. Kein separater Content je Plattform — gleicher Kern, andere Verpackung.

---

## Tool 1: Opus Clip

**Was es ist:** KI-Tool das aus einem langen Video automatisch die besten Clip-Momente erkennt und als Shorts exportiert.

**Wie es funktioniert:**
1. Video-URL (YouTube) oder Upload → Opus Clip analysiert
2. KI erkennt Hook-Momente, Storytelling-Peaks, starke Aussagen
3. Exportiert 3–7 Clips in 9:16, automatisch gecroppt
4. Auto-Captions (funktioniert auf Deutsch, aber Review nötig)
5. Direkter Export zu TikTok, YouTube Shorts, Instagram

**Preis:** ~$20–50/Monat (Starter reicht für MAJER-Anfangsphase)

**MAJER-Workflow mit Opus Clip:**
```
1. Longform-Video auf YouTube hochladen (oder Upload zu Opus Clip)
2. Opus Clip URL eingeben
3. KI wählt 3–7 beste Clips → Review (5 Min)
4. Clips exportieren → Titles anpassen
5. In Buffer hochladen und schedulen
```

**Limitation:** Opus Clip versteht Kontext nicht vollständig. KI-ausgewählte Clips brauchen manuelles Review — vor allem bei inhaltlich präzisen Bildungsthemen.

## Tool 2: CapCut (Desktop + Mobile)

**Was es ist:** Kostenloses Video-Editing-Tool mit starken AI-Features.

**Wichtigste Features für MAJER:**
- **Auto-Caption:** Erkennt gesprochenen Text, erzeugt Untertitel (für DE gute Qualität, Review nötig)
- **Teleprompter:** Text scrollt auf dem Display während Aufnahme → kein Auswendiglernen
- **Templates:** Tausende Vorlagen für Shorts (Vorsicht: nicht alle passen zu MAJER-Brand)
- **Auto-Reframe:** Schneidet 16:9 Video automatisch auf 9:16 zu
- **Beat Sync:** Musik-Schnitt automatisch auf Beats

**MAJER-Nutzung:**
```
CapCut für:
- Rohschnitt von kurzen Videos (<5 Min)
- Auto-Caption auf MAJER-Videos
- Teleprompter bei Aufnahmen
- Schnelle Reels/TikTok Formatierung

NICHT für:
- Longform-Editing (besser: DaVinci Resolve free)
- Komplexe Effekte (zu aufwändig)
```

**Preis:** Kostenlos (mit Pro-Features die optional sind)

---

## Tool 3: Buffer

**Was es ist:** Social-Media-Scheduling-Tool. Einmal hochladen, alle Plattformen zeitversetzt posten.

**Verbundene Plattformen:** TikTok, YouTube, Instagram, LinkedIn, Facebook, Pinterest, Mastodon

**MAJER-Workflow:**
```
1. Clips aus Opus Clip exportieren
2. In Buffer hochladen (1× pro Plattform, oder Cross-Post)
3. Caption für jede Plattform anpassen (TikTok ≠ LinkedIn)
4. Posting-Zeit für jede Plattform festlegen
   - TikTok: 18–21 Uhr (Kernzielgruppe)
   - YouTube: 14–17 Uhr
   - LinkedIn: Mo–Mi, 8–10 Uhr
5. Buffer postet automatisch
```

**Preis:** Free (3 Channels) oder ~$15/Monat (Essentials, unbegrenzte Posts)

**Alternative:** Publer (mehr Features, ähnlicher Preis)

---

## Tool 4: DaVinci Resolve (kostenlos, für Longform)

**Was es ist:** Professionelles Video-Editing-Tool, komplett kostenlos.

**Wann nutzen:** Für YouTube Longform-Videos (>5 Min) wo mehr Kontrolle nötig ist.

**MAJER-Nutzung:** Montage, Cuts, B-Roll einbauen, Farbkorrektur. Lernkurve: 2–3 Stunden für Grundlagen.

---

## Vollständiger Repurposing-Workflow für MAJER

```
AUFNAHME (30 Min)
│   Longform 16:9 + Optional nochmal vertikal 9:16 für beste Short-Qualität
│
▼
UPLOAD (5 Min)
│   Longform → YouTube (unlisted zunächst)
│
▼
OPUS CLIP (10 Min aktiv, 20 Min Warten)
│   URL eingeben → KI analysiert → 3–7 Clips exportieren
│
▼
REVIEW & CAPTIONS (15 Min)
│   Captions prüfen (CapCut oder direkt in Opus Clip)
│   Titles für jeden Clip schreiben
│
▼
BUFFER SCHEDULING (10 Min)
│   Alle Clips + Longform in Buffer
│   Captions je Plattform anpassen
│   Posting-Zeiten einstellen
│
▼
LONGFORM LIVE (manuell: Thumbnail + Beschreibung)
│   YouTube Longform von unlisted → public setzen
│
▼
TRACKING (nächster Tag, 5 Min)
    Buffer-Analytics: Was hat funktioniert?
    Beste Clips notieren → nächste Woche ähnlichen Hook versuchen
```

**Gesamt-Zeitaufwand nach Setup:** ~60–90 Minuten pro Longform-Video inkl. 3–5 Shorts
